Font name:         Kosovo Target BC
Designer:          Bart Claeys
First release:     16 April 1999
Characters:        a-z, A-Z, 0-9, !$%&()+.,:;=

No punctation marks included.

!	->	plane
$	->	American flag
%	->	destroy - attack
&	->	plane-earth symbol
(	->	war stamp
)	->	army stamp
+	->	target symbol
.	->	word bold: "target"
,	->	word normal: "target"
:	->	NATO symbol
;	->	Yugoslavia symbol
=	->	USA peace corps


Kosovo Target BC is a combination of
Courier Bold and a target symbol created
with Coreldraw 8.
The other graphix are all of Coreldraw.
